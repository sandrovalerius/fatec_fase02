A Folha de Estilo

Podemos usar uma folha de estilo de forma constante, geralmente relacionado à tag dentro de uma estrutura.

Dessa forma, economizamos programação de estilo, podendo aproveitar melhor.

No entanto, ao fazer uma programação de estilo complexa para uma página, se estiver interna, teria que copiar e colar para outra página, e se houvesse alguma mudança, teria que fazer a correção nas duas páginas!

Neses casos, vamos usar o método External Style Sheet.
